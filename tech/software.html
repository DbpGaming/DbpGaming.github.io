Software i currently use on my artix install.

Oksh

OpenBSD Korn Shell, great user shell, augmented with Busybox Ash

Dash

fast script execution

Doas

simple sudo replacement

sopen

script i made to replace xdg-open.

sf

script i made to quickly open files.

sfm

terminal file manager i have my own fork of.

st

simple term emu, i forked.

sfeed

simple rss feed fetcher.

dmenu

simple menu for launching applications etc, have my own fork.

Links2

A lynx like Web browser.

Toybox VI

really simple vi clone with just enough features.

Toybox CP/MV/RM

simple basic commands.

dvtm

A multiplexer inspired by DWM

BVI

Binary VI, VI like hex editor.

Htop

A process/task manager.

pfetch

A simple system info Utility, forked.

promptless

simple posix script prompt, my own fork.

pick

Option selection.

Screenshooter

Screenshot tool, basically the same as Snipping Tool on windows

MPV

MPV is video and audio player forked from mplayer, can be used as a music player.

SXIV

Simple X Image Viewer.

Zathura

Document Viewer, use mostly for manga.

7zFM

Peak File Archival Performance.

MultiMC

Minecraft Launcher, supports basically any modpack possible.

PPSSPP

Really accurate and featured PSP emulator.

Dolphin

Decently Accurate but not perfect, and insanely featured GCN/WII emulator.

Mupen64Plus

N64 emulator, minimal, supports plugins, use GlideN64.
