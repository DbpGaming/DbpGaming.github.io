forks and custom software.

sopen

Simple POSIX script to handle file extensions, made for plumbing.

sf

select/simple file, can open files and navigates directorys using dmenu or pick. is a POSIX script.

sfm

Fork of sfm with sopen and keybinding changes.

sxiv

Fork of sxiv without AA, with better zooming, image looping, and saner defaults.

st

Fork of ST with multiplexer support, zoom changes, plumbing using sopen, gruvbox and more.

dvtm

dvtm fork with some code length optimizations.

sfeed

Fork of sfeed with a default dmenu script, and edits to be used with cron.

dmenu

dmenu fork with gruvbox colors, centered, case insensitive.

dwm

dwm fork with gruvbox colors, multiple layouts, fullscreen mode.

pfetch

Fork of pfetch with ascii, wm, and every OS but linux, freebsd, and openbsd support removed. reduces it from 1800 lines, to roughly 550.

promptless

simple posix script prompt, works with oksh and busybox ash.
